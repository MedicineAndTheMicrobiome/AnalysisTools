These steps will extract the nodes (hierachy) and names from two files at
NCBI.


Steps:

1.) Go to: ftp://ftp.ncbi.nlm.nih.gov/pub/taxonomy/
2.) Extract the taxdump.tar.gz file (or whichever taxdump file is easier for you to decompress)
3.) Keep the nodes.dmp and names.dmp file
4.) run extract_nodes_and_names.csh to generate nodes.tsv and names.tsv
